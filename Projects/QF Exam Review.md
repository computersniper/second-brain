## Open-Book Note Template

For each chapter, compress content into:

- Core definition: the concept in one exam-ready sentence.
- When to use: which question or model needs this concept.
- Formula / model: only include formulas that help answer or explain.
- Financial analogy: how the physical idea maps to finance.
- Confusion trap: what not to mix it with.
- Answer template: 2-5 English sentences that can be copied into an exam answer.

## Daily Review Loop

1. Pick one unfinished chapter.
2. Extract formulas, models, diagrams, and key English terms.
3. Add `when to use` and `confusion trap`.
4. Append the highest-value material to [[QF Open-Book Exam Sheet]].
5. End by writing 3 likely exam questions.
